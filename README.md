# NAS 音乐播放器

基于 Flutter 的 Android 音频播放器，通过 WebDAV 协议连接 NAS（飞牛OS / 群晖 / 威联通等），支持音乐和有声书的流式播放。

## 安装

### 方式一：下载 APK 安装

从 [Releases](../../releases) 下载对应架构的 APK（`app-arm64-v8a-release.apk` 适用于大多数现代手机），传到手机安装。

安装前需在手机 **设置 → 安全** 中允许「安装未知来源应用」。

### 方式二：自行构建

```bash
flutter pub get
flutter build apk --split-per-abi --release
```

APK 生成在 `build/app/outputs/flutter-apk/` 目录下。

> 环境要求：Flutter SDK >= 3.3.0、Java 17、Android SDK。

## 使用说明

### 1. 添加 NAS 连接

首次打开 APP 会看到引导页，点击「添加连接」进入配置页：

| 字段 | 说明 | 示例 |
|------|------|------|
| 名称 | 自定义连接名称（可选） | `家里的NAS` |
| 地址 | WebDAV 服务地址，支持 HTTP/HTTPS | `http://192.168.1.100:5005` |
| 用户名 | NAS 登录账号 | `admin` |
| 密码 | NAS 登录密码 | — |
| 基础路径 | WebDAV 起始目录（可选） | `/音乐` |

填写后点击右上角「测试连接」验证地址和账号是否正确。验证通过后点击「保存」。

> **飞牛OS**：在「文件管理 → 设置 → WebDAV」中开启服务。
> **群晖**：在「套件中心 → WebDAV Server」中启用。
> **威联通**：在「控制台 → 应用服务 → WebDAV」中启用。

### 2. 浏览文件

添加连接后进入文件浏览页：

- **目录导航**：点击文件夹进入子目录，顶部面包屑可快速跳转回上级
- **排序切换**：右上角按钮切换按名称 / 修改时间 / 文件大小排序
- **下拉刷新**：在顶部下拉重新加载当前目录
- 页面只显示音频文件（.mp3 / .flac / .wav / .aac / .m4a / .ogg / .wma）和文件夹

### 3. 播放音频

点击任意音频文件开始播放，进入播放页面：

- **播放/暂停**：底部中央按钮
- **上一首/下一首**：左右两侧按钮（根据当前队列模式行为不同）
- **拖动进度条**：跳转到任意位置
- **快进/快退**：两侧 10 秒快进/快退按钮（步长可在设置中调整）

#### 播放模式

点击左下角模式按钮切换：

| 模式 | 图标 | 行为 |
|------|------|------|
| 列表顺序 | `↻` | 按队列顺序播放，播完后停止 |
| 列表循环 | `↻₁` | 队列播完后从头循环 |
| 单曲循环 | `↻¹` | 当前曲目循环播放 |
| 随机播放 | `⇄` | 随机选择下一首 |

#### 播放队列

点击播放页右上角队列图标查看和管理队列：

- 查看当前队列中的所有曲目
- **左滑**项目可移出队列（删除）或移除队列中的连接
- 点击队列中的曲目直接切换到该曲目播放

### 4. 迷你播放器

从播放页返回其他页面后，底部会出现迷你播放器栏：

- 显示当前播放曲目名称
- 播放/暂停按钮
- 点击迷你播放器返回完整播放页

### 5. 后台播放与锁屏控制

播放开始后，即使切换到其他 APP 或锁屏，音频仍会继续播放。

- **通知栏**：显示当前曲目名称，提供播放/暂停、上一首、下一首按钮
- **锁屏**：锁屏界面同样显示媒体控件
- **耳机控制**：支持蓝牙/有线耳机的播放/暂停按钮

> 通知栏左滑可停止播放并关闭通知。部分手机需在系统设置中允许该 APP 的「通知」权限。

### 6. 定时停止

在播放页面点击右下角定时器图标设置定时：

- **5 / 10 / 15 分钟**：倒计时结束后自动暂停播放
- **播完当前**：当前曲目播放完毕后自动停止（不播放下一首）
- **关闭定时**：取消已设置的定时

定时进行中，顶部会显示剩余时间。定时触发后会弹出提示。

### 7. 播放进度记忆

APP 自动记忆每个文件的播放位置：

- **自动保存**：播放过程中定期保存进度，切换页面或退出 APP 时也会保存
- **跳过开头/结尾**：文件开头 < 5 秒和结尾 > 10 秒的位置不会被保存（防止从结尾重播）
- **恢复提示**：下次打开同一文件时，弹窗询问「从上次位置继续播放？」还是「从头开始」
  - 倒计时 5 秒，超时自动从上次位置继续
  - 切换连接后，进度记忆按不同 NAS 独立存储

### 8. 设置

从浏览页或播放页右上角进入设置：

- **默认播放速度**：0.5x ~ 2.0x，适合有声书加速
- **快进/快退步长**：5 秒 / 10 秒 / 15 秒 / 30 秒
- **主题模式**：浅色 / 深色 / 跟随系统
- **管理连接**：查看、编辑、切换或删除 NAS 连接

### 9. 管理多个连接

支持保存多个 NAS 连接：

- **添加连接**：在连接管理页点击「+」
- **切换连接**：在连接列表点击目标连接，自动切换并刷新浏览页
- **编辑连接**：左滑连接项 → 编辑，修改地址或密码
- **删除连接**：左滑连接项 → 删除（至少保留一个连接）

## 技术栈

Flutter · Riverpod · just_audio · audio_service · WebDAV · SQLite · go_router

## 项目结构

```
lib/
├── core/           # 基础设施：数据库、WebDAV 客户端、音频服务
├── features/       # 功能模块：连接、浏览、播放、定时、进度、设置
└── shared/         # 共享数据模型
docs/
├── design/         # 架构与模块设计文档
└── dev/            # 开发状态跟踪
```
